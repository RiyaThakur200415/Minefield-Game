# 💣 Cyber Deception Minefield

Cyber Deception Minefield is an interactive browser-based game developed using HTML, CSS, and JavaScript. The game challenges players to navigate a grid while avoiding hidden cyber traps (mines). Inspired by the classic Minesweeper concept, the project introduces a cybersecurity theme where players must detect threats, avoid deception points, and safely clear the field.

## 📌 Project Overview

Cybersecurity professionals often use deception techniques to detect and mislead attackers. This project gamifies that concept by creating a minefield where hidden threats represent cyber attacks and deceptive traps.

The objective is simple:

- Navigate the grid.
- Avoid hidden mines.
- Use clues to identify danger zones.
- Clear the board without triggering a threat.

The game combines logical thinking, pattern recognition, and strategic decision-making.

---

## 🎮 Gameplay

### Objective

Reveal all safe cells while avoiding hidden cyber threats.

### Rules

- Click on a cell to reveal it.
- Safe cells display information about nearby threats.
- Mines represent cyber attacks or deceptive traps.
- Revealing a mine ends the game.
- Clear all safe cells to win.

## 🎨 Game Components

### Minefield Grid

The game board consists of multiple cells, some containing hidden cyber threats.

### Safe Cells

Reveal information about nearby mines to help players make strategic decisions.

### Threat Cells

Represent hidden cyber attacks. Clicking one results in a game over.

### Victory Condition

Players win by revealing every safe cell without triggering a threat.
